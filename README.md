## 🌱 커밋 컨벤션

| `feat: ~ ` | 새로운 기능 추가 |
| --- | --- |
| `fix: ~ ` | 수정에 대한 커밋 |
| `refactor: ~ ` | 리팩터링에 대한 커밋 |
| `design: ~ ` | 스타일이나 UI 수정 |
| `docs: ~ ` | 문서 수정 |
| `chore: ~` | 패키지 관련설정 |
